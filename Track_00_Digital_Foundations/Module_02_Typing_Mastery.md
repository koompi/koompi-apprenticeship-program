## Lesson 1: Why Typing Speed Matters

### Typing is Your Foundation

As a future software engineer, you will spend **thousands of hours** typing. Every minute you save per hour adds up to hundreds of hours saved over your career.

```
┌─────────────────────────────────────────────────────────────────────────────┐
│ THE MATH OF TYPING SPEED │
├─────────────────────────────────────────────────────────────────────────────┤
│ │
│ Slow Typer (15 WPM) vs. Fast Typer (60 WPM) │
│ │
│ To write 1000 words: │
│ Slow: 67 minutes Fast: 17 minutes │
│ Difference: 50 minutes saved! │
│ │
│ If you write 1000 words daily for a year: │
│ Slow typer: 407 hours │
│ Fast typer: 102 hours │
│ TIME SAVED: 305 hours = 12+ full days! │
│ │
└─────────────────────────────────────────────────────────────────────────────┘
```

### Beyond Speed: Accuracy

Speed without accuracy is useless. If you make many mistakes, you spend time fixing them.

**Goal**: Fast AND accurate typing

## Lesson 2: Proper Hand Position (Touch Typing)

### The Home Row

**Touch typing** means typing without looking at the keyboard. Your fingers learn where each key is through practice.

The **home row** is where your fingers rest:

```
┌─────────────────────────────────────────────────────────────────────────────┐
│ KEYBOARD HOME ROW │
├─────────────────────────────────────────────────────────────────────────────┤
│ │
│ ┌───┬───┬───┬───┬───┬───┬───┬───┬───┬───┬───┬───┬───┬──────────┐ │
│ │ ` │ 1 │ 2 │ 3 │ 4 │ 5 │ 6 │ 7 │ 8 │ 9 │ 0 │ - │ = │ Backspace│ │
│ ├───┴─┬─┴─┬─┴─┬─┴─┬─┴─┬─┴─┬─┴─┬─┴─┬─┴─┬─┴─┬─┴─┬─┴─┬─┴─┬────────┤ │
│ │ Tab │ Q │ W │ E │ R │ T │ Y │ U │ I │ O │ P │ [ │ ] │ \ │ │
│ ├─────┴┬──┴┬──┴┬──┴┬──┴┬──┴┬──┴┬──┴┬──┴┬──┴┬──┴┬──┴┬──┴────────┤ │
│ │ Caps │ A │ S │ D │ F │ G │ H │ J │ K │ L │ ; │ ' │ Enter │ │
│ ├──────┴─┬─┴─┬─┴─┬─┴─┬─┴─┬─┴─┬─┴─┬─┴─┬─┴─┬─┴─┬─┴─┬─┴───────────┤ │
│ │ Shift │ Z │ X │ C │ V │ B │ N │ M │ , │ . │ / │ Shift │ │
│ ├────┬───┴┬──┴─┬─┴───┴───┴───┴───┴───┴──┬┴───┼───┴┬──────┬─────┤ │
│ │Ctrl│Alt │Win │ Space Bar │Alt │ Fn │ Ctrl │Arrow│ │
│ └────┴────┴────┴────────────────────────┴────┴────┴──────┴─────┘ │
│ │
│ HOME ROW KEYS (where fingers rest): │
│ │
│ Left Hand: A S D F Right Hand: J K L ; │
│ (pinky)(ring)(mid)(index) (index)(mid)(ring)(pinky) │
│ │
│ The F and J keys have small bumps — feel them without looking! │
│ │
└─────────────────────────────────────────────────────────────────────────────┘
```

### Finger Assignment

Each finger is responsible for specific keys:

```
LEFT HAND RIGHT HAND
═════════ ══════════

Pinky: A, Q, Z, 1, ` Pinky: ;, ', P, [, ], /, 0, -, =
Ring: S, W, X, 2 Ring: L, O, ., 9
Middle: D, E, C, 3 Middle: K, I, ,, 8
Index: F, G, R, T, V, B, 4, 5 Index: J, H, U, Y, M, N, 6, 7

THUMBS: Both thumbs share the Space Bar
```

### Why This Matters

Using the correct fingers:

- Is faster once you learn
- Reduces hand strain
- Allows you to type without looking
- Becomes automatic with practice

## Lesson 5: Typing Khmer Characters

### Switching Keyboard Layouts

Your KOOMPI computer can type in both English and Khmer. You'll need to switch between them.

**Common shortcut to switch:** `Alt + Shift` or `Super + Space` (depends on your settings)

### Khmer Keyboard Layout

The Khmer keyboard follows a specific layout. Practice is essential!

```
┌─────────────────────────────────────────────────────────────────────────────┐
│ KHMER TYPING TIPS │
├─────────────────────────────────────────────────────────────────────────────┤
│ │
│ 1. PRACTICE COMMON WORDS FIRST │
│ • ខ្មែរ (Khmer) │
│ • កម្ពុជា (Cambodia) │
│ • សួស្តី (Hello) │
│ │
│ 2. LEARN VOWEL COMBINATIONS │
│ • Khmer has many vowel diacritics │
│ • Practice combining consonants with vowels │
│ │
│ 3. PRACTICE SPECIAL CHARACTERS │
│ • Subscript consonants │
│ • Dependent vowels │
│ │
│ 4. USE FOR CUSTOMER COMMUNICATION │
│ • You'll type in Khmer when helping Cambodian customers │
│ • Practice professional phrases │
│ │
└─────────────────────────────────────────────────────────────────────────────┘
```

### When to Use Each Language

| Situation | Language |
|-----------|----------|
| Writing code | English (always) |
| Technical documentation | English |
| Customer chat (Cambodian) | Khmer |
| Internal team communication | Your choice |
| Learning materials | Usually English |

## Progress Milestones

Track your Words Per Minute (WPM) progress:

| Level | WPM | Status |
|-------|-----|--------|
| Beginner | 0-15 WPM | Just starting |
| Learning | 15-25 WPM | Building basics |
| **Minimum Goal** | **30 WPM** | **Required for Module completion** |
| Competent | 30-45 WPM | Good progress |
| Proficient | 45-60 WPM | Above average |
| Expert | 60+ WPM | Excellent |

**Note**: Accuracy should always be **95%+**. Speed comes with practice.
