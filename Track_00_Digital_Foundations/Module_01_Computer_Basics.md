## Lesson 2: Introduction to KOOMPI OS

### What is an Operating System?

An **Operating System (OS)** is the main software that controls your computer. It:

- Manages all the hardware
- Runs other programs
- Provides a way for you to interact with the computer

### KOOMPI OS is Linux-Based

**Linux** is a family of operating systems that is:

- ğŸ†“ **Free** â€” No cost to use
