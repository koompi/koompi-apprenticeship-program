## Lesson 4: Essential Keyboard Shortcuts

### Why Learn Keyboard Shortcuts?

Using the keyboard is **faster** than using the mouse. Professional developers use keyboard shortcuts constantly. Learning them now will make you more efficient.

### The Most Important Keys

- IMPORTANT KEYS
- Ctrl (Control)
- â€¢ Used with other keys for shortcuts
- â€¢ Example: Ctrl + C = Copy
- Alt (Alternate)
- â€¢ Used for menu shortcuts
- â€¢ Example: Alt + Tab = Switch windows
- Shift
- â€¢ Makes letters UPPERCASE
- â€¢ Modifies other shortcuts
- Enter
- â€¢ Confirms/submits
- â€¢ Starts a new line when typing
- Esc (Escape)
- â€¢ Cancels current action
- â€¢ Closes menus
- Tab
- â€¢ Moves to next field
- â€¢ Adds indentation
- Backspace vs Delete
- â€¢ Backspace: Deletes character BEFORE cursor
- â€¢ Delete: Deletes character AFTER cursor

### Essential Shortcuts to Memorize

| Shortcut | Action | Use Case |
|----------|--------|----------|
| `Ctrl + C` | **Copy** | Copy selected text or files |
| `Ctrl + X` | **Cut** | Cut (move) selected text or files |
| `Ctrl + V` | **Paste** | Paste what you copied/cut |
| `Ctrl + Z` | **Undo** | Undo your last action |
| `Ctrl + Y` | **Redo** | Redo what you undid |
| `Ctrl + S` | **Save** | Save your work |
| `Ctrl + A` | **Select All** | Select everything |
| `Ctrl + F` | **Find** | Search for text |
| `Alt + Tab` | **Switch Windows** | Jump between open programs |
| `Alt + F4` | **Close Window** | Close the current program |

### Practice: Shortcuts Exercise

**Try this sequence:**

4. Copy it (`Ctrl + C`)
5. Paste it three times (`Ctrl + V`, `Ctrl + V`, `Ctrl + V`)
