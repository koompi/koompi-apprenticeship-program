# Track 00: Digital Foundations

## KOOMPI Apprenticeship Program

---

## Track Overview

Before building software, you need strong digital foundations. This track ensures every apprentice has the essential skills to succeed.

**Duration**: 4-8 weeks

**This is the starting point for all apprentices!**

---

## Why Digital Foundations?

**Building a Solid Foundation:**

| Without Foundations | With Foundations |
|---------------------|------------------|
| Slow typing = slow coding | Fast typing = productive |
| Lost files = lost work | Organized = efficient |
| Fear of terminal = limited | Terminal comfort = powerful |
| Computer confusion = frustration | Computer mastery = confidence |

*Invest time here → Save time later*

## What You'll Learn

### Module 01: Computer Basics

- Understanding your KOOMPI computer
- Navigating the desktop environment
- Essential keyboard shortcuts
- System settings and customization

### Module 02: Typing Mastery

- Touch typing technique
- Programming symbols practice
- Khmer typing
- Speed and accuracy goals

### Module 03: Linux Terminal

- Why use the terminal
- Navigation commands (cd, ls, pwd)
- File operations (cp, mv, rm, mkdir)
- Terminal productivity

### Module 04: File Management

- Folder structure best practices
- Naming conventions
- Project organization
- Backup strategies

---

## Track Completion Requirements

To earn your **Digital Citizen Certificate**:

- [ ] Complete all 4 modules
- [ ] Typing speed: 30+ WPM minimum (40+ recommended)
- [ ] Comfortable navigating terminal
- [ ] Projects folder organized correctly
- [ ] Mentor verification

2. **Practice typing daily** — It pays off enormously
3. **Use terminal by choice** — The more you use it, the faster you get
4. **Organize from day one** — Good habits start now
