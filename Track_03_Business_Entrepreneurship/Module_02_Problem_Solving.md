3. Then ask for help with context


## Lesson 7: Asking Smart Questions

### Before You Ask

- [ ] Have I tried to solve it myself?
- [ ] Have I searched for solutions?
- [ ] Can I clearly explain the problem?
- [ ] Can I show what I've tried?

### How to Ask

Bad question:

```
"It doesn't work. Help!"
```

Good question:

```
"I'm trying to center a div horizontally. I've tried margin: auto 
and text-align: center, but it's still aligned left. Here's my code
[code snippet]. I'm using Chrome on Windows. What am I missing?"
```

### The XY Problem

Sometimes you ask about your attempted solution (Y) instead of the actual problem (X).

```
Bad: "How do I get the last 3 characters of a filename?"
(You want to check file extension, but asking about substring)

Better: "How do I check if a file is a PDF?"
(Now everyone can suggest the best approach)
```
