## How to Use This Level

### For Each Module

1. **Read** the content carefully
2. **Type** all code examples (don't copy-paste!)
3. **Experiment** by changing the code
4. **Complete** the exercises
5. **Build** the mini-project
6. **Verify** with your mentor

### Folder Setup

Create your HTML learning folder:

```bash
cd ~/projects/learning
mkdir -p html-fundamentals
cd html-fundamentals
```
