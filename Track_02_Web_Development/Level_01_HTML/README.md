# Level 2.1: HTML Fundamentals

## Level Overview

**HTML (HyperText Markup Language)** is the foundation of every website. In this level, you will learn to build the structure of web pages.

**Duration**: 4-6 weeks

**Modules in this Level:**

1. Getting Started with Web Development
2. HTML Document Structure
3. Text Content & Lists
4. Links & Navigation
5. Images & Media
6. Tables & Forms
7. Project: Personal Bio Page

## How to Use This Level

### For Each Module

1. **Read** the content carefully
2. **Type** all code examples (don't copy-paste!)
3. **Experiment** by changing the code
4. **Complete** the exercises
5. **Build** the mini-project
6. **Verify** with your mentor

### Folder Setup

Create your HTML learning folder:

```bash
cd ~/projects/learning
mkdir -p html-fundamentals
cd html-fundamentals
```
