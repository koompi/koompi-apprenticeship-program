## CSS Changes Everything

```css
/* Just a few lines of CSS... */
body {
 font-family: 'Arial', sans-serif;
 line-height: 1.6;
 color: #333;
 background-color: #f5f5f5;
}

h1 {
 color: #2c3e50;
 text-align: center;
}

/* ...transforms your entire page! */
```
