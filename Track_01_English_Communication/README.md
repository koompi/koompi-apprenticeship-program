## Daily English Practice

### Build These Habits

| Time | Activity | Duration |
|------|----------|----------|
| Morning | Read one tech article in English | 15 min |
| Work | Respond to customers in English | Throughout |
| Coding | Read documentation in English | As needed |
| Evening | Write daily log in English | 15 min |

### English in Your Coding

When you code, you encounter English constantly:

- Variable names: `userName`, `isLoggedIn`
- Error messages: "Cannot read property of undefined"
- Documentation: How to use functions
- Commit messages: "Fix bug in login form"
