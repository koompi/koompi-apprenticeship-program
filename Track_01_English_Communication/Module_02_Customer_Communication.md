## Lesson 1: Communication Principles

### The KOOMPI Way

- CUSTOMER COMMUNICATION PRINCIPLES
- 1. BE FRIENDLY
- • Start with a greeting
- • Use the customer's name
- • Show you care
- 2. BE HELPFUL
- • Focus on solving their problem
- • Provide clear information
- • Go the extra mile
- 3. BE PROFESSIONAL
- • Use proper grammar
- • Avoid slang
- • Stay calm always
- 4. BE QUICK
- • Respond promptly
- • Don't leave customers waiting
- • If you need time, tell them

## Lesson 2: Greeting Customers

### Opening Messages

**Formal:**

```
Good morning! Thank you for contacting KOOMPI. How may I help you today?

Hello! Welcome to KOOMPI support. I'm happy to assist you.
```

**Friendly:**

```
Hi there! Thanks for reaching out. What can I help you with?

Hello! Great to hear from you. How can I assist you today?
```

### Using Customer's Name

When they introduce themselves:

```
Hi, my name is Sokha...

Response:
Hi Sokha! Thanks for reaching out. How can I help you today?
```

### Time-Appropriate Greetings

| Time | Greeting |
|------|----------|
| Morning (6am-12pm) | Good morning! |
| Afternoon (12pm-5pm) | Good afternoon! |
| Evening (5pm-9pm) | Good evening! |
| Anytime | Hello! / Hi there! |

## Lesson 3: Handling Inquiries

### Product Questions

**Customer:**

```
How much is the KOOMPI E13?
```

**Response:**

```
Thank you for your interest in the KOOMPI E13! 

The KOOMPI E13 is priced at $299. It comes with:
- 13.3" Full HD display
- 8GB RAM
- 256GB storage
- KOOMPI OS (Linux-based)

Would you like more information about any specific features?
```

### Availability Questions

**Customer:**

```
Do you have it in stock?
```

**Response:**

```
Yes, we currently have the KOOMPI E13 in stock and ready for delivery!

Would you like to place an order? I can help you with that.
```

**If not available:**

```
I apologize, but the KOOMPI E13 is currently out of stock. 
We expect new stock to arrive in approximately 2 weeks.

Would you like me to notify you when it's available again?
```

## Lesson 4: Problem Solving

### When Customers Have Issues

**Step 1: Acknowledge**

```
I'm sorry to hear you're experiencing this issue. Let me help you resolve it.
```

**Step 2: Gather Information**

```
Could you please provide more details about the problem?
- When did this issue start?
- What exactly happens?
- Have you tried restarting?
```

**Step 3: Provide Solution**

```
Thank you for that information. Based on what you've described, 
I recommend trying the following:

1. First, restart your laptop
2. If the issue continues, try [specific solution]

Please let me know if this resolves the issue.
```

### Common Phrases for Problems

| Situation | Phrase |
|-----------|--------|
| Acknowledging | "I understand how frustrating this must be." |
| Apologizing | "I apologize for the inconvenience." |
| Asking for patience | "Thank you for your patience while I look into this." |
| Following up | "I'm checking on this now. Please give me a moment." |
| Offering alternatives | "Alternatively, you could try..." |

## Lesson 5: Handling Complaints

### Stay Calm and Professional

**Angry customer:**

```
Your product doesn't work! I want my money back!
```

**NEVER respond with:**

```
 That's not my fault.
 You must be doing something wrong.
 Calm down.
```

**ALWAYS respond with:**

```
 I sincerely apologize for this experience. I understand your frustration.

Let me help make this right. Could you please tell me:
1. What specific issue are you experiencing?
2. When did you purchase the product?

I'll do everything I can to resolve this for you quickly.
```

### HEAT Method for Complaints

```
H - Hear them out (let them explain fully)
E - Empathize (show you understand)
A - Apologize (sincerely)
T - Take action (solve the problem)
```

### Example Resolution

```
Customer: My laptop screen is cracked and I just bought it!

Response:
I'm very sorry to hear about your screen. That must be really frustrating, 
especially with a new purchase.

Let me help you with this right away. First, I'd like to understand more:
- When did you receive the laptop?
- Was there any visible damage to the packaging?

Depending on your answers, we have several options:
1. Warranty replacement
2. Repair service
3. Exchange for a new unit

I'll make sure we find the best solution for you.
```

## Lesson 7: Closing Conversations

### After Solving the Issue

```
I'm glad I could help resolve this for you!

Is there anything else I can assist you with today?
```

### Ending Positively

```
Thank you for choosing KOOMPI! 

If you have any more questions in the future, 
don't hesitate to reach out. Have a great day! 
```

### Asking for Feedback

```
I hope I was able to help you today. 

We're always looking to improve—if you have a moment, 
we'd appreciate any feedback on your experience.

Thank you again, and have a wonderful day!
```

## Common Response Templates

### Inquiry Template

```
Hi [Name]!

Thank you for your interest in [product/service].

[Answer their question]

[Additional helpful information]

Is there anything else you'd like to know?

Best regards,
[Your name]
KOOMPI Support
```

### Problem Resolution Template

```
Hi [Name],

I apologize for the inconvenience you've experienced.

[Acknowledge the specific issue]

Here's what I recommend:
1. [Step 1]
2. [Step 2]
3. [Step 3]

Please let me know if this resolves the issue. 
If not, I'm here to help further.

Best regards,
[Your name]
KOOMPI Support
```
